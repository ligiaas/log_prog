Tipos de dados

Alguns tipos de dados que uma variável pode conter:

  • String => texto
  • Número Inteiro => 10
  • Número Real => 10.5
  • Boleano => true/false
  • Array => [lista]
  • Objetos => relógio{props: metodos{}}
  • null
  • undefined
  • NaN
